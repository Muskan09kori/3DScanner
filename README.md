# 3D Scanner AR Application

A professional 3D scanning application built with Unity 2022, featuring AR visualization using Vuforia Engine.

## Features

- Real-time 3D scanning with AR visualization
- Vuforia AR Engine integration (v11.4.4)
- Support for GLB, FBX, and OBJ model formats
- Professional UI/UX
- Cross-platform support (Android, iOS, Windows)

## Technologies

- **Engine:** Unity 2022.3.62f2 LTS
- **AR Framework:** Vuforia Engine 11.4.4
- **Language:** C#

## Installation

1. Clone the repository:
```bash
   git clone https://github.com/Muskan09kori/3DScanner.git
```

2. Open in Unity 2022.3 LTS or higher

3. Configure Vuforia license key:
   - Window > Vuforia Engine > Open License Manager
   - Add your Vuforia license key

4. Build for your target platform (Android/iOS/Windows)

## System Requirements

- Unity 2022.3 LTS or higher
- Android 8.0+, iOS 13.0+, or Windows 10+
- Minimum 4GB RAM
- Device camera required for AR
